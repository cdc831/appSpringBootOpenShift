Que joraca pasa!

Simple control de Logs para saber que esta pasando. d;P

Preciona el boton para listar los logs ingresados o borra el historial, tambien podes usar un cliente rest como Insomnia, Postman, etc.


Url para Rest

  • [GET] /logger/list - Listar todos los logs
  • [POST] /logger/add - Para agregar un Logs en formato Json || Ejemplo: { "clave": "String", "valor": "String", "opcional": "String" }
  • [DELETE] /logger/delall - Para borrar todos los logs
